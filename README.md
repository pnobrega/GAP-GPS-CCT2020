## Setup

First clone this project to your local enviroment

```
$ git clone https://github.com/pnobrega/GAP-GPS-CCT2020.git
```

To run this project, install it locally using npm:

```
$ cd ../GAP-GPS-CCT2020
$ npm install
$ npm start
```
